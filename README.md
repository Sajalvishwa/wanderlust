# 🌍 WanderLust – Travel Listing Web Application

WanderLust is a full-stack web application that allows users to explore, create, and manage travel listings. It provides a seamless experience for discovering new places, adding listings, and interacting with user-generated content.

---

## 🚀 Features

* 🔐 User Authentication (Login / Signup)
* 🏡 Create, Edit & Delete Listings
* 📍 Location-based Listings with Map Integration
* 🖼️ Image Upload using Cloudinary
* 💬 Flash Messages & Error Handling
* ⭐ Clean and Responsive UI
* 🔎 Explore Listings from Different Locations

---

## 🛠️ Tech Stack

**Frontend:**

* HTML, CSS, Bootstrap
* EJS (Templating Engine)

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB (Mongoose)

**Other Tools & Services:**

* Cloudinary (Image Hosting)
* Passport.js (Authentication)
* Render (Deployment)

## ⚙️ Installation & Setup

1. Clone the repository



2. Navigate to project folder

```
cd wanderlust
```

3. Install dependencies

```
npm install
```

4. Create a `.env` file and add:

```
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_KEY=your_api_key
CLOUDINARY_SECRET=your_api_secret
DB_URL=your_mongodb_url
SESSION_SECRET=your_secret
```

5. Run the app

```
npm start
```
